If deployment is FAILED or CRASHED:
────────────────────────────────────

Common Issue #1: Wrong Start Command
Fix: In Railway Settings > Deploy > Start Command
Set to: cd backend && npm install && npm run build && npm start

Common Issue #2: Missing package.json start script
Fix: Check backend/package.json has:
"scripts": {
  "start": "node dist/index.js"
}

Common Issue #3: TypeScript errors
Fix: Check build logs for "error TS..."
Need to fix TypeScript errors in code

Common Issue #4: Database connection
Fix: Make sure DATABASE_URL is set correctly
Should be the Supabase connection string

========================================
TROUBLESHOOTING
========================================

Problem: Can't find the "+" button
Solution: Look for "New" or "Deploy" button in Railway dashboard

Problem: GitHub repo not showing
Solution: You may need to authorize Railway in GitHub settings

Problem: Build fails
Solution: Check the logs in Railway, ensure all env vars are set

Problem: "Cannot connect to backend"
Solution: 
- Make sure backend deployed successfully
- Check NEXT_PUBLIC_API_URL in Vercel
- Wait 2-3 minutes for Vercel redeploy
